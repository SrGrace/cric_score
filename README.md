# cric_score
Cricket Score form Cricbuzz



Install : 

          1) Using pip

             pip install cric_score
             
          2) Using git
          
            git clone https://github.com/sourav0007/cric_score.git
            cd cric_score
            python setup.py install
            
            
            
            
Usage :
      
      Simply run the following command 
      __  circ_score
